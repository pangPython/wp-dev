== 安装 ==

1. 登陆 WordPress 管理员后台（仪表盘），从侧边栏进入 “外观” -> “主题”，点击“添加” -> “上传主题”，选择主题文件“boke-1.zip”，并点击“现在安装”。

2. 安装完成后，点击“启用”。

3. 从侧边栏进入 “外观” -> “自定义”，设置你的主题。

4. 从侧边栏进入 “外观” -> “小工具”，使用你所需的小工具。
